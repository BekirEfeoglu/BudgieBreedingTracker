# Facebook Group Outreach Playbook

Amaç Facebook kuş topluluklarında spam yapmadan, grup kurallarına uygun şekilde Budgie Breeding Tracker görünürlüğü kazanmaktır. Bu kanal sadece izinli topluluk paylaşımı, admin onayı ve faydalı içerik üzerinden yürütülür.

## Kesin Kurallar

- Toplu otomatik paylaşım yapılmaz.
- Grup kuralları okunmadan paylaşım yapılmaz.
- Reklam, tanıtım, link veya uygulama paylaşımı yasak olan gruplara post atılmaz.
- Admin onayı gerekiyorsa onay yazılı olarak alınmadan post hazırlanmaz.
- Aynı metin kısa sürede çok sayıda gruba kopyalanmaz; her grup için bağlam ve dil uyarlanır.
- Satış/sahiplendirme odaklı gruplarda uygulama tanıtımı yerine sadece izin istenir.
- Sağlık, tedavi veya veterinerlik iddiası yazılmaz.
- Grupta biri iletişim istemezse takip mesajı gönderilmez.

## Hedef Grup Tipleri

| Öncelik | Grup tipi | Yaklaşım |
| --- | --- | --- |
| P1 | Muhabbet kuşu üretimi, eşleme, kuluçka ve yavru takibi konuşulan gruplar | Admin izniyle faydalı takip şablonu + uygulama tanıtımı |
| P1 | Kuş üreticileri / kafes kuşları üreticileri grupları | Üretim kaydı ve çoklu çift takibi odağı |
| P2 | Muhabbet kuşu yardım, bakım ve soru-cevap grupları | Reklam yerine ücretsiz kuluçka takip tablosu ve eğitim içeriği |
| P2 | Papağan, kanarya, kafes kuşları genel grupları | Uygulama kapsamının muhabbet kuşuna odaklı olduğu net belirtilir |
| P3 | Sahiplendirme grupları | Sadece admin izin mesajı; doğrudan tanıtım yapılmaz |

## Uygunluk Kontrolü

Bir grup paylaşım için uygun sayılmadan önce aşağıdaki alanlar doldurulur:

| Kontrol | Uygun değer |
| --- | --- |
| Grup bağlantısı | Facebook grup URL'si |
| Grup konusu | Muhabbet kuşu, kuş üretimi, kuluçka, yavru takibi veya kafes kuşları |
| Üyelik durumu | Sayfa/profil gruba katıldı veya public katılımcı paylaşımı açık |
| Reklam kuralı | Tanıtım serbest, admin izniyle serbest veya haftalık tanıtım başlığı var |
| Admin onayı | Onay alındıysa tarih ve ekran görüntüsü notu |
| Paylaşım tipi | Eğitim postu, izinli tanıtım, ücretsiz şablon veya soru-cevap katkısı |
| Sonuç | Beklemede, izin istendi, onaylandı, reddedildi, paylaşıldı |

```text
Uygulama; kuş, eş, yumurta, kuluçka, yavru, soyağacı ve genetik takibini düzenli tutmaya yardımcı oluyor. Grubunuzda admin onayıyla kısa ve faydalı bir tanıtım paylaşımı yapmam uygun olur mu?
```

```text
Anladım, grup kuralına saygı duyuyorum. Paylaşımı kaldırabilir veya tekrar paylaşmayabilirim. İyi günler.
```

## Haftalık İş Akışı

Pazartesi:

- 10 yeni grup adayı bul.
- Kuralları ve reklam izin durumunu kontrol et.
- Uygun görünenleri `facebook-group-outreach-tracker.csv` dosyasına ekle.

Salı:

- En uygun 5 grubun adminine izin mesajı gönder.
- Cevapları takip tablosunda güncelle.

Çarşamba-Perşembe:

- Onay gelen gruplar için metni grup bağlamına göre kısalt.
- Paylaşım öncesi son kez kural ve link kontrolü yap.

Cuma:

- Paylaşılan postların yorumlarını kontrol et.
- Sadece kullanıcı soru sorarsa yanıt ver.
- Link tıklamalarını UTM üzerinden haftalık dashboard'a işle.

Hafta sonu:

- Yeni post atmak yerine yorum/soru-cevap katkısı yap.
- Aynı gruba haftada birden fazla tanıtım postu atma.

## Durdurma Kriterleri

Aşağıdaki durumlardan biri olursa o grupta paylaşım durdurulur:

- Admin veya moderatör izin vermez.
- Post kaldırılır.
- Grup üyeleri reklamdan rahatsız olduğunu belirtir.
- Aynı grup içinde link paylaşımı spam olarak işaretlenir.
- Konu sağlık/tedavi tavsiyesine kayar.
