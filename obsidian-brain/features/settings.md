# Feature: settings

**Purpose**: App settings — theme, data export, backup, account management.

## Key Screens

- Settings list
- Data export (PDF/Excel)
- Account deletion flow
- Security settings (MFA enrollment)

## Export

- PDF: `pdf ^3.10.0`
- Excel: `excel ^4.0.0`
- Shared via OS share sheet (`share_plus`)

## Security Settings

- MFA enable/disable
- Change password
- Linked OAuth providers

## Developer Menu Access

5-tap on Settings header title → developer/experimental features menu. See [[patterns/feature-flags]].
